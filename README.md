# Automação de Tarefas com Python

Projeto desenvolvido durante a Jornada Python da Hashtag.

## Descrição
Automação do processo de cadastro de produtos utilizando Python, permitindo executar tarefas repetitivas de forma automática.

## Tecnologias utilizadas
- Python
- PyAutoGUI
- Pandas (se tiver usado)

## Objetivo
Reduzir tempo gasto em tarefas manuais e aumentar a produtividade.

## Como executar o projeto

1. Clone o repositório
2. Instale as dependências necessárias
3. Execute o arquivo:

python automacao.py
